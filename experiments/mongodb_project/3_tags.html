In this week's experiment, I implemented tag selection which is quite popular in today's websites.

The tags are just a list of tags, and displayed as list. The main challenge is to switch between different css classes when user select or cancel a tag. And this is implemented by using ng-class ternary operation ng-class="expression ? class1 : class2".

Here the expression is an angular js function. It'll return true or false according to current selected tag.

Following code is html code to display tags and their styles according to current status.

            <div class="col-sm-10, task-tag-list">
                <ul>
                    <li ng-repeat="tag in timerCtrl.tags">
                        <a ng-click="timerCtrl.chooseL1Tag(tag)"
                           ng-class="timerCtrl.isTagActivated(tag.tagName) ? 'task-tag-activated' : 'task-tag-unactivated'">
                            {{tag.tagName}}</a>
                    </li>
                </ul>
            </div>

Following is the js code. If current display tag is the selected tag or subtag, it'll return true, which leads to the use of task-tag-activated css style class.

                    this.activatedTag = {}
                    this.activatedSubTag = ''

                    this.isTagActivated = function(tn) {
                        if (tn == this.activatedTag.tagName || tn == this.activatedSubTag) {
                            return true
                        } else {
                            return false
                        }
                    }
