Documentation

In this experiment, I tried to implement a multi-tab page from scratch using AngularJS..

There are 2 tabs in this page and each tab is associated with one info page. I used AngularJS's ng-show to show the specific page based on the variable activeTab.

Also, ng-class is used to render the tab when user switch from one another. One thing I learned from this experiment is, if the class name has - inside, then be sure to surround it with single quotes.

Code Snippet

Following code shows how the tab swith is done. tabs.click(1) will set the activeTab to 1, and this change will further impact ng-class of the tab and ng-show of the content.

                   <ul class="list-inline list-unstyled">
                       <li><a  ng-class="{'activeTab': tabs.show(1)}" href ng-click="tabs.click(1)">Documentation</a></li>
                       ....
                   </ul>

                    <div class="bg-success">
                        <div ng-show="tabs.show(1)">
                            <h2>Documentation</h2>

                            ....
                        </div>
