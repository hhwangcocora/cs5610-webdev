Experiment 04: Implements tabs using HTML/CSS and jQuery (without bootstrap)

Documentation

In this experiment, I implemented my homepage's tabs using html/css and jQuery, without using bootstrap.

All the css styles are designed from scratch. The tab switching is implemented by jQuery: add active class to clicked a, and show the div whose id is clicked href name. All others are hided.

Following is the css for the HOMEPAGE tab. It defines the font color, size, family, and also margin and padding. When the a is hovered or active, we'll display a bottom border.

                .nav-main {
                    color: #ffffff;
                    font-size: 1.3em;
                    font-family: Arial;
                    padding: 10px 0px 10px 0px;
                    margin-right: 40px;
                }

                a:hover.nav-main, a:active.nav-main, .nav-main.active {
                    border-bottom: solid 5px #0088ff;
                }

Following is the jQuery code for tabs switching. On document loading ready, it'll first find the div with id homePage, show its content and hide all the siblings. When user click on the tab, it'll first get the href of the tab, which is also the id of the content. Then, the code show/hide tabs and change/remove the content.

                jQuery(document).ready(function() {
                    $('#homePage').fadeIn(500).show().siblings().hide();
                    jQuery('#navigation a').on('click', function(e)  {
                        var currentAttrValue = jQuery(this).attr('href');

                        // Show/Hide Tabs
                        jQuery(currentAttrValue).fadeIn(500).show().siblings().hide();

                        // Change/remove current tab to active
                        jQuery(this).addClass('active').siblings().removeClass('active');

                        e.preventDefault();
                    });
                });
