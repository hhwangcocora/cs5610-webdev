Experiment 3: customize icon

Documentation

In this experiment, I customized the icon for the web site. You can see that the icon of the tab is changed to be a red tomato.

It is achieved by setting shortcut icon to the icon image.

Code Snippet

First of all, place the icon image on some public accessible path. Then put following line in the html page. By setting short icon, the icon will then be displayed on all four kinds of browsers.

        <link rel="shortcut icon" href="favicon.ico" />
