Documentation

In this experiment, I tried drawing line chart and bar chart using Chart.js library.

Charts are very important part of a web page, to show statistics to user in a very straightforward way. In some cases, it's even better than tables.

The Chart.js provides very succinct yet powerful api to draw different charts. This experiments only shows two kinds of them, however there are also charts like pie, radar, polar area, etc. And it also provides various options to customize the chart.

Code Snippet

Following code puts a canvas on the page which will be used to draw the chart.

        <canvas id="myLineChart" width="500" height="300"></canvas>

Following is the javascript code to draw the line chart. It firstly gets the myLineChart canvas, then create a Chart object using that context and provides the data.

        var data = {
        labels: ["Boston", "New York", "San Fransisco", "Washington DC", "Seattle", "Dallas"],
        datasets: [
            {
                label: "Average Temperature",
                fillColor: "rgba(0,220,220,0.2)",
                strokeColor: "rgba(0,220,220,1)",
                pointColor: "rgba(0,220,220,1)",
                pointStrokeColor: "#fff",
                pointHighlightFill: "#fff",
                pointHighlightStroke: "rgba(220,220,220,1)",
                data: [65, 59, 80, 81, 56, 55]
            },
            {
                label: "Average Humility",
                fillColor: "rgba(151,187,205,0.2)",
                strokeColor: "rgba(151,187,205,1)",
                pointColor: "rgba(151,187,205,1)",
                pointStrokeColor: "#fff",
                pointHighlightFill: "#fff",
                pointHighlightStroke: "rgba(151,187,205,1)",
                data: [28, 48, 40, 19, 86, 27, 90]
            }
        ]
    };

    var ctx = $('#myLineChart').get(0).getContext('2d')
    var myLineChart = new Chart(ctx).Line(data, null);
