Code Snippet

Following code shows how to use Angular JS route module to control navigation to different pages. Here $routeProvider is the builtin module name, and within the config() we configures different html pages and their controller for different navigation path. On html page, we use ng-view to embed those pages.

    app.config(['$routeProvider',
    function($routeProvider){
        $routeProvider.
            when('/home', {
                templateUrl: 'partials/home.html',
                controller: 'HomeController',
                controllerAs: 'hc'

            }).
            ...
            otherwise('/home')
    }]
)

    <nav class="nav navbar-collapse">
        <a href="#/home">Home</a> |
        <a href="#/documentation">Documentation</a> |
        <a href="#/codesnippet">Code Snippet</a> |
        <a href="#/references">References</a>
    </nav>
    <br/>
    <div ng-view></div>

The navigation path is implemented by using a simple ol with class setting to breadcrumb

    <ol class="breadcrumb">
        <li><a href="#/home">Home</a></li>
        <li class="active">Code Snippet</li>
    </ol>
