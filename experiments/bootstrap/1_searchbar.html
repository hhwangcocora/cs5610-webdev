In this experiment, I used bootstrap css to stylish a typical search form. (Similar to the advanced search page in our myNEU course search. )

First of all, the whole form is put inside container to make it stay in middle of screen. Then, I used col-sm- to layout labels and inputs in the grid system. control-label makes lables align to right. col-sm-2-off moves the header and buttons 2 columns right, to make them align with inputs.

For the credit range, I used input-group-addon class so that the 4 elements can concatenate with each other.

The two buttons are set to different types, search is primary and reset is default.

Finally, to control the input length, I put them inside a div with class set to col-sm-x. The inputs themselves are with class form-control.

Code Snippet

Following is about the two buttons at bottom. They are moved 2 columns right, and set to different button types.

            <button class="btn btn-primary col-sm-offset-2" type="submit">Search</button>
            <button class="btn btn-default" type="reset">Reset</button>

Following is the credit range: 

            <div class="input-group col-sm-4">
                <div class="input-group-addon">From</div>
                <input type="number" class="form-control" id="fromCredit" placeholder="credit">
                <div class="input-group-addon">To</div>
                <input type="number" class="form-control" id="toCredit" placeholder="credit">
            </div>
