Documentation

In this experiment, used bootstrap carousel plugin to implement a slide show. Slide show is a very important element in commercial website.

The carousel is composed with 3 parts, the carousel-indicators which are a list of spots, carousel-inner are the contents displayed, in this case, they are pictures and captions. carousel-controls are the left and right angle brackets.

Since the picture is too small to fill the space, I need to set explicitly the width to 200px, such that there is no empty space left. Also, the left and right angle brackets can only display correctly by linking to the bootstrap css file instead of downloading to local.

Code Snippet

Code about carousel

                <div id="carousel-cloths" class="carousel slide" data-ride="carousel">
                    <ol class="carousel-indicators">
                        ...
                    </ol>
                    <div class="carousel-inner" role="listbox">
                        <div class="item active center-block">
                            <img src="../../images/experiments/slideshow1.jpg" alt="cloth 1" class="img-responsive img-thumbnail">
                            <div class="carousel-caption">
                                <p>Lorem ipsum dolor sit amet, consectetur</p>
                            </div>
                        </div>
                        ...
                    </div>
                    <a class="left carousel-control" href="#carousel-cloths" role="button" data-slide="prev">
                        <span class="glyphicon glyphicon-chevron-left" aria-hidden="true"></span>
                        <span class="sr-only">Previous</span>
                    </a>
                    <a class="right carousel-control" href="#carousel-cloths" role="button" data-slide="next">
                        <span class="glyphicon glyphicon-chevron-right" aria-hidden="true"></span>
                        <span class="sr-only">Next</span>
                    </a>
                </div>

Code to set picture size

                .fix-slide-window {
                    width: 200px;
                }
