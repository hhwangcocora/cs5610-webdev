Documentation

In this experiment, I practise bootstrap table and popover plugin.

The table is stylished with classes table, table-hover, and table-striped. table-hover makes the row highlisted when mouse hover on it. table-strped hightlights odd rows.

The course numbers are buttons with popovers. When user click it, a popover appears to show the course description. On next click, it disappears. This is implemented by using data-toggle=popover, with data-trigger=focus. Be noted that, popover need to be explicitly enabled. See following code snippet.

Code Snippet

Following code is how the popover effect work

                <td><button type="button" class="btn btn-link" data-toggle="popover" data-trigger="focus" title="CS5610 - Web Development"
                            data-content="Description: ......">CS5610</button></td>

Following script need to be added explicitly to make popover work.

                $(function () {
                    $('[data-toggle="popover"]').popover()
                })
