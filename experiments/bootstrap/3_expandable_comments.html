Documentation

In this experiment, I used bootstrap css and javascript to create a comment section.

I put article, comment button, and comment into different rows, such that when comment expand, it won't change the layout of other elements. The comment section is set to be col-sm-6.

To have the collapse effect, I set comment button's data-toggle to collapse, and href to the comments section. This will call into javascript's collapse function when user click the button

Inside the comments section, I put text input field and button inside a form-inline form so that they can stay in same line

One import lesson I learned is, always load jquery before bootstrap js code.

Code Snippet

Following code is how the collapse effect happen for the comments button

                <div class="row">

                    <a class="btn btn-primary btn-sm" data-toggle="collapse" href="#comments" aria-expanded="false"
                       aria-controls="comments">
                        Comments
                    </a>
                    ...
                </div>
                <div class="row">
                    <div class="collapse col-sm-6" id="comments">
                        <div>
                            <ul class="list-unstyled">
                                ...
                                <li>
                                    <form class="form-inline">
                                        <div class="form-group">
                                            <input type="text" class="form-control" id="yourComment" placeholder="Comment">
                                            <button class="btn btn-default" type="submit">Submit</button>
                                        </div>
                                    </form>
                                </li>
                            </ul>
                        </div>
                    </div>
                </div>
