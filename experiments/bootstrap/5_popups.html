Change password

Experiment 5: Popups using bootstrap

Documentation

In this experiment, I added a profile icon at the right of navigation bar just like most website did. It has a dropdown list of operations (currently only "Change password" is active). If user click "Change password", there will be modal poped up asking for old password and new password.

By using navbar-right, the profile div is pushed right. The dropdown list is implemented using dropdown plugin.

The popup window is implemented using bootstrap's dialog modal. It typically consisted of 3 parts, modal-header, modal-body and modal-footer. Header is the title, body contains the password input fields, and footer has the buttons to submit the change or discard. These are put inside a modal-dialog avoids opaque background, and modal-dialog which makes the dialog to be smaller and compact.

Code Snippet

How to open a modal:

                <li><a href="#passwordModel" data-toggle="modal">Change password</a></li>

Part of the modal code:

        <div class="modal fade" id="passwordModel" tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">
            <div >
                <div class="modal-content">
                    <div class="modal-header">
                        <button type="button" class="close" data-dismiss="modal" aria-label="Close"><span aria-hidden="true">×</span></button>
                        <h4 class="modal-title" id="myModalLabel">Change password</h4>
                    </div>
                    <div class="modal-body">
                        <form class="form-horizontal">
                            ....
                        </form>
                    </div>
                    <div class="modal-footer">
                        <button type="button" class="btn btn-default" data-dismiss="modal">Cancel</button>
                        <button type="button" class="btn btn-primary">Change password</button>
                    </div>
                </div>
            </div>
        </div>
