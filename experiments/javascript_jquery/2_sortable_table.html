Documentation

In this experiment, I implemented a sortable table using javascript and jQuery.

Sortable table is a very user friendly feature. It helps user to sort the information in order for better understanding and interpretation. It is especially helpful if the page contains a lot data and attributes, since user can locate certain information quickly by putting them in order.

I practised on how to use jQuery wildcard selector and javascript sorting function in this experiment.

Code Snippet

Following code shows how the jQuery wildcard selector works. It will get all elements whose id matches tr-

            function emptyTable() {
                $('[id^=tr-]').empty() // use wildcards to remove all table rows
            }

Following code is to sort an array of objects. The function(a, b) is the sorting function for the course object. In this experiment, it uses course number to sort object.
