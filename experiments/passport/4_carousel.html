Documentation

In this experiment, I added a 3 image carousel on the homepage as most website will do.

The carousel is done by using Bootstrap carousel component. Besides the html, the carousel need to be explicitly enabled in javascript code. You can find the html code in home.html.

Code Snippet

Following code is the javascript code to set active the carousel. The inerval: 4000 means it'll take 4 seconds to display each image, and cycle: true means the images will be displayed in cycle.

                app.controller('homeController', function($scope) {

                    $('.carousel').carousel({
                        interval: 4000,
                        cycle: true
                    })

                })
