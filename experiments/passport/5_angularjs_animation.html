Angular js animation during routing and hide/show tags

Go to project page(admin:123)

Try switch between different tabs and also add/remove tags.


Documentation

In this experiment, I tried add animations when switching between different tabs and when user add/remove/click tags.

The animation is implemented using angular js ng-enter and ng-enter-active which tells how the appearance of the object should be look like.

Any objects carried with the class will animate when appear on the page.

Code Snippet

Following code the css code to define the animation. The second section is pre-animation, and the third section is post-animation.

            .show-animation.ng-enter {
                -webkit-transition: .3s linear all; /* Safari/Chrome */
                -moz-transition: .3s linear all; /* Firefox */
                -o-transition: .3s linear all; /* Opera */
                transition: .3s linear all; /* IE10+ and Future Browsers */
            }

            .show-animation.ng-enter{
                /* The animation preparation code */
                opacity: 0;
            }

            .show-animation.ng-enter.ng-enter-active {
                /* The animation code itself */
                opacity: 1;
            }

Then ng-view div is with class show-animation

            <div class="show-animation" ng-view></div>
